Software (and Library) Carpentry aims to help researchers (and those that serve them) get their work done in less time and with less pain by teaching them basic research computing skills. This hands-on workshop will cover basic concepts and tools, including program design, version control, data management, and task automation. Participants will be encouraged to help one another and to apply what they have learned to their own research problems.

Please make sure to see the setup instructions below. For the first workshop you will only need bash, for the second git, and for the third python. The extra information provided is just that, extra. You will not need abything besides bash, git and python for the workshops, but the other material may be useful.

Please email Tim Norris if you have any questions or concerns about the setup requirements.

For more information on what we teach and why, please see our paper "Best Practices for Scientific Computing".
